Codex Skill

Xiaohongshu Video Backup

把小红书视频备份、重试失败链接、转写成同名文字笔记，收成一套可重复使用的工作流，现在也可以直接作为 Mac 桌面工具来跑。

查看 Skill 查看 README

它解决什么问题

  • 批量处理小红书笔记链接，不用手动一条条下载。
  • 根据解析到的小红书标题自动命名视频文件。
  • 遇到中断或超时后，可以按失败列表继续补下。
  • 把下载好的 MP4 转成同名转写和 Markdown 内容笔记。

现在可以当 Mac App 用

  • 仓库里已经带了桌面界面。
  • 双击 `launch_mac_app.command` 就能启动。
  • 窗口里可以直接粘贴多个 URL，一行一个。
  • 也支持直接粘贴整段分享文案，程序会自动提取里面的链接。
  • 也可以继续选择链接文件、视频目录、笔记目录。
  • 运行日志会显示在窗口底部。
  • 也可以进一步构建成标准的 `.app` 包。

最简单的用法

  1. 下载仓库到本地。
  2. 双击 `launch_mac_app.command`。
  3. 直接粘贴多个小红书链接，一行一个。
  4. 在窗口里点下载和生成笔记。

1. 下载视频

用 `greenvideo.cc` 做解析，脚本读取 URL 列表，下载视频，并把失败项单独记录下来。

2. 断点补跑

如果电脑关机、批处理超时、个别链接解析慢，只需要拿失败文件继续跑，不必从头开始。

3. 生成笔记

用 Whisper 转写中文视频，输出同名 `.txt` 和 `.md`，便于归档、检索和二次改写。

Mac 上这样启动

git clone https://github.com/Jas0nla/xiaohongshu-video-backup-skill.git
cd xiaohongshu-video-backup-skill
chmod +x ./launch_mac_app.command
./launch_mac_app.command

生成 .app

python3 mac_app/build_app.py
open "dist/Xiaohongshu Video Backup.app"

也可以继续用命令行

python3 skill/scripts/download_videos.py \
  --urls-file /abs/path/urls.txt \
  --output-dir /abs/path/xhs_videos \
  --failures-file /abs/path/download_failures.json

python3 skill/scripts/generate_notes.py \
  --videos-dir /abs/path/xhs_videos \
  --transcripts-dir /abs/path/xhs_transcripts \
  --notes-dir /abs/path/xhs_notes

先准备什么

  • 最省事的方式：直接把多个 URL 粘贴进窗口输入框。
  • 一个 `urls.txt` 文件，一行一个小红书链接。
  • 本机可用的 `python3`、`curl`、`npx`、`whisper`。
  • 如果你在 Codex 里用，还需要本机有 `playwright` skill。
