Codex Skill

Xiaohongshu Video Backup

把小红书视频备份、重试失败链接、转写成同名文字笔记，收成一套可重复使用的工作流。

查看 Skill 查看 README

它解决什么问题

  • 批量处理小红书笔记链接，不用手动一条条下载。
  • 根据解析到的小红书标题自动命名视频文件。
  • 遇到中断或超时后，可以按失败列表继续补下。
  • 把下载好的 MP4 转成同名转写和 Markdown 内容笔记。

1. 下载视频

用 `greenvideo.cc` 做解析，脚本读取 URL 列表，下载视频，并把失败项单独记录下来。

2. 断点补跑

如果电脑关机、批处理超时、个别链接解析慢，只需要拿失败文件继续跑，不必从头开始。

3. 生成笔记

用 Whisper 转写中文视频，输出同名 `.txt` 和 `.md`，便于归档、检索和二次改写。

示例命令

python3 skill/scripts/download_videos.py \
  --urls-file /abs/path/urls.txt \
  --output-dir /abs/path/xhs_videos \
  --failures-file /abs/path/download_failures.json

python3 skill/scripts/generate_notes.py \
  --videos-dir /abs/path/xhs_videos \
  --transcripts-dir /abs/path/xhs_transcripts \
  --notes-dir /abs/path/xhs_notes
